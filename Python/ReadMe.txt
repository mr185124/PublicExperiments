Usage: 
1. Extract zip file to a folder such as C:/Support/JREFinder
2. Open up a command prompt and navigate to the folder you saved in Step 1.
3. Enter JREFinder.exe
4. JREFinder will contact the AdoptOpenJDK website and save the most recent release in LatestVersion.txt in the same path as the exe.
5. Subsequent runs will either return "No new versions found" or update LatestVersion.txt.

Version: 1.0 (2021.03.06)

Source code: https://github.com/mr185124/PublicExperiments/blob/main/Python/JREFinder.py
